# Professor-de-Ciberseguranca
Utilizei o NotebookLM para armazenar 50 fontes de dados para uma Super IA Especialista em ensinar segurança digital.

Após as aulas do professor Felipão da DIO. Conheci um novo método de treinar I.A, NotebookLM do Google capaz de armazenar dados inseridos pelo próprio usuário. Mesmo com um limite máximo de 50 fontes no pacote Free, a ferramenta ainda continua sensacional oferecendo diversos formatos de apresentação das informações de saída (vídeo, áudio, apresentação, mapa metal...).

Com isso, consegui reunir 50 fontes diferentes de diversos assuntos do universo de T.I. Chamado de Dr. Cenoura o chat acumula conhecimentos de Redes de computadores, do básico ao avançado.

Tais conhecimentos são [Redes de Computadores - Segurança da Informação - Cibersegurança - Pentest e "OWASP" - Hacking Ético - Linux - Kali Linux - Analista de SOC - Metodologias, Técnicas e atuação do SOC - Framework NIST].

Dados extraídos de fontes como [vídeo do youtube, pdf, arquivo e recomendações minhas].
